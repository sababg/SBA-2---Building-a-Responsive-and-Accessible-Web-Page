# Front end mentor Co. – Responsive Web Page

This project is a responsive, accessible multi-section webpage built for the fictitious company **Front end mentor Co.**  
It showcases semantic HTML, modern CSS layout techniques, and accessibility best practices.

## Features

- Semantic HTML structure (`header`, `nav`, `main`, `section`, `article`, `footer`)
- Responsive layout using CSS Grid and Flexbox
- Mobile-friendly design with media queries
- Accessible navigation with ARIA roles and labels
- Accessible forms with proper labels and validation feedback
- Consistent styling using CSS custom properties (variables)

## Technologies Used

- HTML5
- CSS3
  - Flexbox
  - Grid
  - Media Queries
  - CSS Variables
- ARIA for accessibility support
